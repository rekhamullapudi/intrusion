# Network Intrusion Detection using Big Data ML

This project implements a scalable machine learning based intrusion detection system using Apache Spark.

## Workflow

1. Data ingestion and cleaning
2. Feature engineering
3. Model training and comparison
4. Scalability analysis
5. Tableau visualization dashboards

## Dataset

UNSW-NB15 network intrusion dataset containing labeled normal and attack traffic.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosted Trees

## Output

The system predicts whether a network connection is normal or malicious and provides visual analytics dashboards.

## How to Run

Run notebooks sequentially:

1_data_ingestion.ipynb → 2_feature_engineering.ipynb → 3_model_training.ipynb → 4_evaluation.ipynb
